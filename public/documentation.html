How To Install Our Application

Go to https://github.com/mmirabello/377groupproject/.

We are using GitHub as a tool to collaborate and host our source code.

How to Run Our Application

  • Download / clone repository to your local machine
  • Go to project folder on your local machine
  • Do npm install
  • Run node server.js
  • Go to your browser and put in the URL http://localhost:3000/

    • API Process

    The application fetches the speed camera API from the PGC database. The data is then parsed to get coordinates (latitude and longitude), the address, and the posted speed. The coordinates are then used to plot markers on the map.

    Future Plans/Known Bugs

    We plan in the future to implement a solution to one of our information problems. One of our problems was that there was no information readily available online over how many tickets each camera gives. In order to solve this we will have a data entry web form to allow us to crowd source this information from our users. Currently as well our "Help" page does not actually send out any email or message to our developement team. We will implement this to send out an email to the developers.
