How To Install Our Application

Go to https://github.com/mmirabello/377groupproject/. We are using GitHub as a tool to collaborate and host our source code.

How to Run Our Application

  • Step 1.) Download / clone repository to your local machine
  • Step 2.) Go to project folder on your local machine
  • Step 3.) Do npm install
  • Step 4.) Run node server.js
  • Step 5.) Go to your browser and put in the URL http://localhost:3000/
